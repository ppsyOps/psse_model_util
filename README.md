# PSSE Model Utility

This utility reads in PSSE 34 models and provides some utilities for the models.  This project was created to facilitate comparing BES level model changes to MMWG based models (like IDC models) within a specific area (such as MISO or PJM).

## Name
PSSE v34 Model Utility

## Description
This library contains utilities to read a PSSE v34 model (.raw or .seq) file into a Model object.  
The Model contains pd.DataFrames for type of equipment read in from the model (buses, branches, transformers, etc.).
You can also compare models with the "compare" module to find added/removed equipment and identify model changes such 
as branch splits.
